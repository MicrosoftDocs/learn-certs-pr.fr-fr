# Quel email doit être lié à la demande de réduction pour les examens liés aux événements ?

Pour certains événements Microsoft, les participants peuvent bénéficier d'une réduction sur l'examen de certification Microsoft. Selon l'événement, l'email que vous devez entrer pour vérifier les réductions admissibles peut différer. Veuillez utiliser le tableau ci-dessous pour vérifier si l'événement est éligible et les instructions appropriées sur la manière de demander votre remise:

| Evénements | Instructions sur l'adresse électronique à saisir |
| --- | --- |
| [Microsoft Ignite September 2019](/learn/certifications/microsoft-ignite-free-certification-exam-offer?WT.mc_id=msignitethetour2019_akawwlflag_-email-event) | Email que vous avez utilisé pour vous inscrire à l'événement |
| [Microsoft Ignite the Tour 2019-2020](/earn/certifications/microsoft-build-cloud-skills-challenge-2020-free-certification-exam-offer) | Email que vous avez utilisé pour vous inscrire à l'événement  |
| [Microsoft Build May 2020](/apprendre/certifications/microsoft-build-cloud-skills-challenge-2020-free-certification-exament-offre) (Événement virtuel) |Email que vous avez utilisé pour vous inscrire à l'événement  |
| Journée de formation virtuelle Microsoft - Azure Fundamentals | Email que vous avez utilisé pour vous inscrire à l'événement  |
| Journée de formation virtuelle Microsoft - Dynamics Fundamentals | Email que vous avez utilisé pour vous inscrire à l'événement  |
| Journée de formation virtuelle Microsoft – Power Platform Fundamentals |  Email que vous avez utilisé pour vous inscrire à l'événement  |
|| [Microsoft Ignite September 2020](/learn/certifications/microsoft-ignite-cloud-skills-challenge-2020-free-certification-exam) (Virtual Event) | Email utilisé pour relever le Défi des compétences dans le nuage sur l'apprentissage (Email on Learn Profile)) |
